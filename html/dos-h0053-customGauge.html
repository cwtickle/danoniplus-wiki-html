customGauge

使い方

|customGauge=Original::F,Normal::V,Escape::V|

|gaugeOriginal=x,4,60,25$x,3,20,25|
|gaugeNormal=50,70,1,25$x,3,25,25|
|gaugeEscape=0,1,50,100$x,5,30,25|

説明

  • gaugeXと合わせて使用します。
  • カンマ区切りで、ゲージ設定毎の設定を行います。
    「ゲージ名::回復/ダメージ量設定」の組み合わせで設定します。
    回復/ダメージ量設定は、「F」なら矢印数によらず固定、
    「V」なら矢印数により変動します。
  • ライフ制/ノルマ制の区別は各個別のゲージ設定(gaugeX.html)にて行います。
    このため、ノルマ制ながら回復/ダメージ量は固定にするといった設定が可能になります。

補足

  • カスタムゲージを指定した場合は gaugeX の指定 もしくは
    danoni_setting.jsのg_presetGaugeCustomの設定が必須です。
    また、OriginalとNormalがデフォルトゲージで無い場合についても、
    設定が必須となります。ご注意ください。
  • 原則、ライフ設定は difData ではなく gaugeX で行い、
    自動設定されているゲージ設定についても、個別に設定してください。
  • カスタムゲージを設定した場合、その作品全体でゲージ設定が共有されます。
    特定の譜面だけ別のゲージ名を採用することはできません。設定値は譜面毎に設定できます。
  • danoni_setting.jsのg_presetGaugeCustomに追加するゲージ設定を記載することで
    どの作品に対してもカスタムゲージを適用できるようになります。
// ゲージ設定（デフォルト以外）
const g_presetGaugeCustom = {
	// (省略)
	Escape: {
		Border: 70,
		Recovery: 1,
		Damage: 50,
		Init: 70,
	},
};

関連項目

  • difData (*, ★) 譜面情報
  • gaugeX ゲージ設定の詳細
  • maxLifeVal ライフの上限値
  • frzStartjdgUse フリーズアロー開始判定の設定有無
  • frzAttempt フリーズアローヒット時の許容フレーム数
